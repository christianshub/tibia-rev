# x64dbg Cheat Sheet

## Commands

| Keys 	        | Description                                  	|
|--------------	|--------------------------------------------	|
| F2           	| Set software bp                            	|
| F7           	| Step into                                    	|
| F8    	    | Step over                                    	|
| F9            | Run                                           |
| CTRL+A       	| Analyze code (guess functions/calls)       	|
| ALT+E        	| Go to modules                              	|
| Enter        	| Follow execution flow (e.g. follow a call) 	|
